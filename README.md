# بوت رعد — بوت إدارة مجموعات تليجرام

بوت تليجرام عربي خليجي لإدارة المجموعات: رتب، كتم/حظر، ردود مخصصة،
ألقاب متعة، ترحيب، تفعيل/تعطيل، ولوحة تحكم للمطور.

## المتطلبات
- Python 3.10+
- Redis (محلي أو Upstash)
- توكن بوت من [@BotFather](https://t.me/BotFather)

## التشغيل المحلي

```bash
cd bot
python3 -m venv venv
source venv/bin/activate        # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt

cp .env.example .env
# عدّل .env وحط التوكن و DEVP_ID و REDIS_URL

python main.py
```

## متغيرات البيئة (.env)
| المتغير | الوصف |
|---|---|
| `TELEGRAM_BOT_TOKEN` | توكن البوت من BotFather |
| `DEVP_ID` | ايديك في تليجرام (المطور الأساسي) |
| `REDIS_URL` | رابط Redis، محلي `redis://localhost:6379` أو Upstash `rediss://...` |
| `DEV_GROUP_ID` | (اختياري) مجموعة إشعارات المطور |

## خطوات التفعيل في مجموعة
1. أضف البوت للمجموعة وارفعه أدمن بكل الصلاحيات (إدارة، حذف، تثبيت، دعوة).
2. أرسل `تفعيل` — يرفعك مالكاً ويرفع كل أدمن تليجرام تلقائياً.
3. جرّب: `رفع مدير` (بالرد)، `كتم` (بالرد)، `الادمن`، `اضف رد`.

## أهم الأوامر
- **الرتب:** رفع/تنزيل (مالك اساسي، مالك، مدير، ادمن، مميز) + تعطيل/تفعيل الرفع
- **الكتم/الحظر:** كتم، الغاء الكتم، كتم عام، حظر عام
- **القوائم:** الادمن، المدراء، الملاك، المميزين، قائمة الكتم
- **الردود:** اضف رد، الردود، مسح رد [كلمة]، اضف ردي، اضف رد عام
- **الأوامر:** اضف امر [الأمر] [النص]، الاوامر المضافه
- **ألقاب المتعة:** رفع كيك، قائمة الكيك، رفع حمار ... إلخ
- **الترحيب:** وضع الترحيب، الترحيب، مسح الترحيب
- **معلوماتي:** اسمي، ايديي، رتبتي، رسائلي، المجموعة
- **لوحة المطور (بالخاص):** /start، الاحصائيات، تعيين اسم البوت [اسم]
